# Algorithms for Apparent Contact Angle Determination  
### Tools for Image-Based Processing

This repository provides base files for determining contact angles using image-based processing methods. It includes tools for synthetic droplet generation, validation of a novel multi-technique experimental device, and practical examples organized by measurement technique.

---

## Repository Content

- **SmartDroplet: Synthetic Droplet Generation & Analysis Toolkit**
- **BoonTap_System Validation**
- **Base Files for Image-Based Contact Angle Determination**

---

## SmartDroplet Toolkit

MATLAB tools for generating synthetic droplet contours, processing droplet shapes, and validating contact angle algorithms.

---

## BoonTap_System Validation

Validation of a multi-technique device for contact angle measurements under HPHT conditions.

---

## Measurement Techniques

- Sessile Drop  
- Captive Bubble  
- Tilted Plate  

---

## Applications

- Surface science
- Wettability studies
- Algorithm validation
